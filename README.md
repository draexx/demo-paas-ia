# Demo PaaS IA

This is a demo project for a Platform as a Service (PaaS) with some AI integrations.

## Project Structure

- `index.html`: The main entry point of the web application, displaying information about the project.
- `LICENSE`: The license file for the project.
- `README.md`: This file, providing an overview and instructions.

## Getting Started

To get started with this demo project, you can deploy it using Vercel.

1.  **Fork this repository:** Click the "Fork" button on the top right of the GitHub repository page.
2.  **Connect to Vercel:** Go to [Vercel](https://vercel.com/) and create an account if you don't have one. Connect your GitHub account to Vercel.
3.  **Import Project:** In Vercel, import your forked repository as a new project. Vercel will automatically detect the `index.html` file and deploy it as a static website.
4.  **Visit your deployed site:** Once deployed, Vercel will provide you with a URL to access your live demo.

You can also run this project locally by opening `index.html` in your web browser.
